# SMART SUGGESTION SYSTEM - LAB 2

Ứng dụng gợi ý trải nghiệm giải trí dựa trên tâm tư người dùng, tích hợp trí tuệ nhân tạo (giả lập) và lưu trữ dữ liệu thời gian thực trên **Google Firebase Firestore**.

## Công nghệ sử dụng
* **Frontend:** Streamlit (Python)
* **Backend:** FastAPI (Python)
* **Database:** Google Firebase (Firestore & Authentication)
* **Server:** Uvicorn
